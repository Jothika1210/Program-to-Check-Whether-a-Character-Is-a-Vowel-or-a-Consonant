# Program-to-Check-Whether-a-Character-Is-a-Vowel-or-a-Consonant

ch=input("enter a character:")
if(ch=='A' or ch=='a' or ch=='E' or ch=='e' or ch=='T'or ch=='i' or ch=='O' or ch=='o' or ch=='U' OR ch=='u'):
    print(ch,"is a vowel")
else:
    print(ch,"is a consonant")

 

OUTPUT
enter a character:e
e is a vowel
